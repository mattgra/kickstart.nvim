# Changelog

## 2025-12-11

### Added
- **render-markdown.nvim**: Pretty markdown rendering in Neovim
- **aerial.nvim**: Symbol outline sidebar for navigating markdown headings and code symbols
  - `<leader>a` - Toggle aerial sidebar
- **Custom plugins import**: Enabled `lua/custom/plugins/` directory for modular plugin configuration
- **Line movement keymaps**: `Alt+j` and `Alt+k` to move lines/selections up and down in normal and visual modes
- **snacks.nvim plugin**: Integrated folke/snacks.nvim with picker functionality
  - `<leader>ff` - Find files
  - `<leader>fg` - Live grep
  - `<leader>fb` - Browse buffers
  - `<leader>fr` - Recent files
- **Plugin import structure**: Added `lua/plugins/` directory with modular plugin configuration
